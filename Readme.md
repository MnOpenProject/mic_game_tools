# 游戏辅助脚本

* 对一些好玩的游戏写一下合理的辅助脚本，并不会破坏游戏的可玩性，只为了弥补一些游戏的小小不足之处

## 目录说明
* 
    * EldenRing/ 目录 -- 请尽量退出游戏到菜单界面，再进行备份和还原游戏记录，否则可能损坏游戏记录文件
    * * 【艾尔登法环】游戏记录备份和还原脚本 -- 请进入该目录下执行 py main_func.py 选择相应的操作即可 
* web 服务启动
    * 可在子目录下执行 py start.py 启动一个web服务，然后访问终端提示的服务地址即可
